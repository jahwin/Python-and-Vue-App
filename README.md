# Desktop app made by Vue and Python

## Project setup
```
npm install
python -m pip install -r requirements.txt
```

### Compiles and hot-reloads for development
```
npm run dev

// Hot reload
npm run watch
```

### Compiles and minifies for production
```
npm run prod
```
